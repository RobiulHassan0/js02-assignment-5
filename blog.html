Important FAQ

What is the Document Object Model (DOM)?
The DOM is a programming interface for web documents. It represents the structure of an HTML or XML page as a tree of objects, allowing JavaScript to access, modify, or delete elements dynamically.
How do you select an element from the DOM?

You can select an element using methods like:

  • document.getElementById('id')
  • document.querySelector('.class' or '#id')
  • document.getElementsByClassName('class')
  • document.getElementsByTagName('tagname')
What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element to handle events from its children using event bubbling. It’s useful because:

  • You avoid attaching multiple listeners.
  • It's efficient for dynamic elements added after the DOM has loaded.
How do you manipulate an element's attributes and styles using the DOM?

You can manipulate attributes using:

  • element.setAttribute('name', 'value')
  • element.getAttribute('name')

And manipulate styles using:

  • element.style.property = 'value'

Example: element.style.color = 'red';
